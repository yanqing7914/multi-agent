# 行为准则

## 我们的承诺

为营造开放、友善的环境，我们承诺：无论年龄、体型、残障、民族、性别认同与表达、经验水平、教育、社会经济地位、国籍、相貌、种族、宗教或性取向，参与本项目的每个人都不受骚扰。

## 我们的标准

**积极行为示例：**

- 使用友善、包容的语言
- 尊重不同观点与经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

**不可接受行为示例：**

- 性化的语言或 imagery，不受欢迎的性关注
- 挑衅、侮辱/贬损评论，人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息
- 其他可合理认定为不专业或不当的行为

## 我们的责任

维护者有义务澄清可接受行为的标准，并对不当行为采取适当、公平的纠正措施。

## 适用范围

本准则适用于项目空间及代表项目或社区公开场合。代表项目包括官方邮件、社交媒体账号，或在线上/线下活动中担任指定代表。

## 执行

可向项目维护者报告滥用、骚扰或其他不可接受行为。所有投诉将被审查并调查，必要时采取回应。维护者有义务对举报人保密。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版，全文见 <https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>。
