# Environment Check

Before multi-agent work, capture:

- Current workspace / cwd.
- Shell and OS.
- Sandbox mode and write permissions.
- Approval policy.
- Network availability.
- Git status and existing user changes.
- Applicable AGENTS.md or repo instructions.
- Available skills/plugins relevant to the task.
- Whether Desktop attachments, screenshots, documents, spreadsheets, presentations, or images are involved.
- Test/build commands available in the project.

If the environment blocks necessary actions, either downgrade the plan or ask the user for confirmation.
